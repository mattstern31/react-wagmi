# Commands

## `init`

Creates configuration file. If TypeScript is detected, the config file will use TypeScript and be named `wagmi.config.ts`. Otherwise, the config file will use JavaScript and be named `wagmi.config.js`.

### Usage

```bash
wagmi init
```

### Options

| Option                | Description                     |
| :-------------------- | :------------------------------ |
| `-c, --config <path>` | `[string]` Path to config file. |

## `generate`

Generates code based on configuration.

### Usage

```bash
wagmi generate
```

### Options

| Option                | Description                                                           |
| :-------------------- | :-------------------------------------------------------------------- |
| `-c, --config <path>` | `[string]` path to config file                                        |
| `-r, --root <path>`   | `[string]` root path to resolve config from                           |
| `-w, --watch`         | `[boolean]` watch for changes (for plugins that support `watch` mode) |
